twitter-on-merb

Description

aggregate edge merb commit rss feeds and send them to twitter @edgemerb.

site => http://merb.kicks-ass.org
This is a merb flat app that gather feeds from edge merb commits and
push the messages to twitter.com/edgemerb.

Dependencies

twitter4r, twitter, nokogiri, shorturl, memcache
